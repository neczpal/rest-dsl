# REST DSL

## Setup

To successfully set up and work with this project, please follow the steps below.

### Prerequisites

*   **IntelliJ IDE**: This project is designed to be used with IntelliJ IDEA.
*   **ANTLR v4 Plugin**: You must install the [ANTLR v4 plugin](https://plugins.jetbrains.com/plugin/7358-antlr-v4) to correctly parse and generate code from the grammar files.
