# Week 8

## Progress:

-   Implemented Model Inheritance:
    -   The DSL now supports model inheritance using the `ChildModel: ParentModel` syntax.
    -   The Java generator creates `sealed interface` for base models and `record` classes for child models, promoting immutability and modern Java practices.
    -   The OpenAPI generator uses `allOf` and a `discriminator` to correctly model polymorphism.

### Example

**RestDSL:**

```rdsl
models {
    User {
        id: Int
        email: String
    }

    Individual: User {
        firstName: String
        lastName: String
    }

    Company: User {
        companyName: String
    }
}
```

**Generated Java:**

```java
public sealed interface User {
    Integer id();
    String email();
}

public record Individual(Integer id, String email, String firstName, String lastName) implements User {}

public record Company(Integer id, String email, String companyName) implements User {}
```

**Generated OpenAPI YAML:**

```yaml
components:
  schemas:
    User:
      type: object
      properties:
        id:
          type: integer
        email:
          type: string
    Company:
      allOf:
      - $ref: "#/components/schemas/User"
      - type: object
        properties:
          companyName:
            type: string
    Individual:
      allOf:
      - $ref: "#/components/schemas/User"
      - type: object
        properties:
          firstName:
            type: string
          lastName:
            type: string
```
